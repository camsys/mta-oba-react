MTA Bus Time has a developer API from day one!

Transit developers, web developers, hobbyists, hackers, or anyone who uses mass transit and has a bit of tech skills - you are part of the solution. MTA Bus Time is built with a real-time developer API that anyone can use for free. We are excited to see what you come up with.

How To Get Involved

Go here to request an API Key. You will receive one within half an hour. While you're waiting, please check out the documentation for the SIRI Real-Time API and the OneBusAway "Discovery" API.

If you have any questions, comments, bug reports, or anything else to say about the MTA Bus Time API, please post them to the MTA Developer Resources Google Group. We look forward to hearing from you. Remember, this is a pilot project, so your input can help shape the future deployments of the MTA Bus Time system.

For information about licensing and about MTA's other developer feeds, please see the MTA's Developer Resources page.
